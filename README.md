# 🧮 Calculadora Java

Projeto desenvolvido em Java com o objetivo de praticar lógica de programação, estruturas condicionais e repetição.

## 📌 Funcionalidades

- Entrada de dois números
- Operações matemáticas básicas (+, -, *, /)
- Validação de divisão por zero
- Execução contínua até o usuário encerrar

## 🛠 Tecnologias
- Java
- Scanner
- switch-case
- do-while

## ▶️ Execução

Clone o repositório:

git clone https://github.com/YguDev/Calculadora-.git

Abra o projeto na IDE (NetBeans) e execute a classe `CalculadoraJava`.
